Purpose

The AuthController.js serves as the primary entry point for all identity-related operations. It acts as an orchestrator between the HTTP request layer and the underlying authService, ensuring validated credentials before session issuance.

Key Functions

  • handleLogin: Validates body schema and initiates JWT generation.
  • handleLogout: Invalidates current token in Redis and clears browser state.
  • refreshSession: Issues a new access token using a valid refresh rotate key.
